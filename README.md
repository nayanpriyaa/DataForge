# DataForge

DataForge is a full-stack data quality and analytics platform that enables users to upload datasets, generate profiling reports, perform data quality analysis, track dataset versions, and analyze dataset characteristics through an intuitive dashboard.

## Features

- JWT Authentication
- Dataset Upload
- Data Profiling
- Data Quality Analysis
- Dataset Analytics
- Dataset Versioning
- Dataset Lineage Tracking

## Tech Stack

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- Pandas

### Frontend
- React
- TypeScript
- Vite
- React Router
- Axios
